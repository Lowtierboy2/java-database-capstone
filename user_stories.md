User Stories
Admin User Stories
- As an Admin, I want to manage user accounts so that doctors and patients can securely access the Smart Clinic system.
- As an Admin, I want to assign system roles (Admin, Doctor, Patient) so that each user has the correct permissions.
- As an Admin, I want to view all registered users so that I can monitor system activity.
- As an Admin, I want to deactivate or remove accounts so that unauthorized users cannot access the system.
- As an Admin, I want to access the Admin Dashboard via Thymeleaf so that I can manage the system through a simple interface.
- As an Admin, I want to oversee appointments and system data so that the clinic runs smoothly.

Patient User Stories
- As a Patient, I want to create an account so that I can access the Patient Dashboard and book appointments.
- As a Patient, I want to log in securely so that my personal and medical information stays protected.
- As a Patient, I want to view available doctors and times so that I can choose the best appointment slot.
- As a Patient, I want to book an appointment through the REST API so that I can schedule visits easily.
- As a Patient, I want to view my upcoming appointments so that I can stay organized.
- As a Patient, I want to cancel or reschedule appointments so that I can manage my schedule.
- As a Patient, I want to view my prescriptions stored in MongoDB so that I can follow my treatment plan.

Doctor User Stories
- As a Doctor, I want to log in through the Doctor Dashboard (Thymeleaf) so that I can access my schedule.
- As a Doctor, I want to set and update my availability so that patients can book appointments at the right times.
- As a Doctor, I want to view my daily and weekly appointments so that I can prepare for consultations.
- As a Doctor, I want to update appointment statuses so that the system reflects completed or missed visits.
- As a Doctor, I want to create and update prescriptions stored in MongoDB so that patients receive accurate treatment information.
- As a Doctor, I want to access patient records via REST APIs so that I can make informed medical decisions.
